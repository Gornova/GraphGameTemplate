## Introduction

GraphGameTemplate is a simple graph-based template for videogames, used as learning and starting point.

## Status

In progress:
* [Sprint 01 completed](SPRINT-01.md)

## Vision

Write a little template to:
* define a graph
* show it using MarteEngine
* ask info from it
